# Local Setup

## Requirements

- Node.js 20+

## First Run

1. `npm install`
2. `npm run bootstrap`
3. `npm run lint`
4. `npm run format:check`
5. `npm test`

## Start Local Workspaces

- `npm run dev`

Expected endpoints:
- Frontend: `http://localhost:5173`
- Backend health: `http://localhost:8080/health`
